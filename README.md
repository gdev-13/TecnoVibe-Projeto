# TecnoVibe Store

Uma página web interativa e moderna para apresentação de produtos tecnológicos, focada em dispositivos Apple. O projeto apresenta um carrossel dinâmico com três produtos principais: Apple Watch Series 10, Air Pods Max e Vision Pro.

## 📋 Descrição

TecnoVibe Store é uma demonstração de uma loja online moderna, construída com tecnologias web básicas (HTML, CSS e JavaScript). O design apresenta um tema futurista com cores ciano e azul, animações suaves e um layout responsivo. O carrossel automático muda de produto a cada 8 segundos, mas também permite navegação manual através de botões e indicadores.

## ✨ Funcionalidades

- **Carrossel Interativo**: Navegação automática e manual entre produtos
- **Design Responsivo**: Adaptável a diferentes tamanhos de tela
- **Animações Suaves**: Transições fluidas entre produtos
- **Indicadores Visuais**: Pontos e números para mostrar o produto atual
- **Tema Futurista**: Estilo cyberpunk com efeitos de brilho e gradientes

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura da página
- **CSS3**: Estilização e animações
- **JavaScript (Vanilla)**: Funcionalidade do carrossel
- **Google Fonts**: Fontes Montserrat e Orbitron

## 🚀 Como Executar

1. Clone ou baixe o repositório
2. Abra o arquivo `index.html` em qualquer navegador web moderno
3. A página será carregada automaticamente

### Pré-requisitos

- Navegador web com suporte a HTML5, CSS3 e JavaScript (Chrome, Firefox, Safari, Edge)

## 🎨 Características do Design

- **Cores Principais**: Azul ciano (#00F7FF) e fundo escuro (#0F0F1E)
- **Fonte Principal**: Montserrat para texto, Orbitron para títulos
- **Efeitos Visuais**: Brilho, gradientes e animações de pulso
- **Layout**: Header fixo com navegação, conteúdo centralizado

## 📱 Responsividade

O projeto é otimizado para visualização em desktop. Para dispositivos móveis, pode ser necessário ajustar o CSS para melhor adaptação.
